# Техническое задание
## 1. Общие сведения
Наименование проекта: "Информационная система для интернет-магазина природных камней".
Исполнитель: Студент Дементьев Фёдор Александрович, обучающийся по направлению 09.02.07, на специальность веб-разработчика.
Заказчик: Преподаватель Московского Колледжа Информационных технологий(МКИТ).
Цели и задачи: Сайт намерен дать возможность для клиентов закупить стройматериалы в виде природных камней для бетонирования и создания фундамента, мощения дорог, создания ландшафта и других целей. 

## 2. Общие требования
- Использование инструментов разработки по выбору разработчика(Редактор кода VisualStudioCode; Языки программирования HTML, JavaScript, веб-фреймворки React, Django; Онлайн-сервис для дизайна Figma;)
- Дизайн должен отображать необходимый функционал в доступной форме для обширной аудитории;
- Временные сроки выставляются по продолжительности семестра, около 8 недель;
- Веб-сайт должен быть адаптирован под маобильные устройства и компьютеры;
